## 実装メモ
### 海岸線の生成
3stepで生成する。
1. 大まかな構造を作成する。
   全体として、メルカトル図法のように、横長のマップにする。
   大陸の大きさもばらつきをいれる。
   左右はトーラス状にして、陸の面積と海の面積比は3:7になるように。
2. 島を作成する。
	島は大陸に被らないように。
	群島のような構造ができるように、偏りを持たせる.
3. フラクタルで細かい構造を追加する。
